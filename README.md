# PrimeSite Studio Website

A modern, professional, and fully responsive website for PrimeSite Studio - a web development agency.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Dark theme with blue accents
- Smooth animations and modern UI
- Smooth scrolling navigation
- Interactive hover effects
- Contact form with Formspree integration
- SEO-optimized structure
- Multiple sections: Hero, Services, About, Portfolio, Why Choose Us, Testimonials, Contact, Footer

## Sections

1. **Hero Section** - Catchy headline and call-to-action buttons
2. **Services Section** - Website Development, UI/UX Design, SEO Optimization, Fast Performance & Hosting
3. **About Section** - Company introduction and core values
4. **Portfolio Section** - Showcase of sample projects
5. **Why Choose Us** - Key advantages and benefits
6. **Testimonials** - Client reviews
7. **Contact Section** - Contact form and information
8. **Footer** - Company info and social links

## Technologies Used

- HTML5
- CSS3 (Custom CSS with modern features)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts (Inter)
- Formspree (for contact form)

## Contact Form Setup

The contact form uses Formspree to send emails. Messages sent through the form will be delivered to: primesitestudio12@gmail.com

To customize the form or set up your own Formspree endpoint:
1. Visit [Formspree.io](https://formspree.io)
2. Create a free account
3. Create a new form
4. Replace the form action URL in index.html with your unique Formspree URL

## Deployment

This website is configured for deployment on GitHub Pages.

## Local Development

Simply open `index.html` in a web browser to view the website locally.
